# Laravel Local Setup & Selenium Automation

## Overview

This repository documents my learning while setting up an existing Laravel application locally and automating its login page using Python Selenium.

## What I Learned

- Setting up a Laravel project
- Installing PHP and Composer
- Configuring XAMPP and MySQL
- Managing Composer dependencies
- Understanding Laravel routing and Blade views
- Browser automation using Selenium WebDriver
- Inspecting HTML elements for automation
- Debugging environment and runtime issues

## Technologies

- PHP
- Laravel
- Python
- Selenium
- MySQL
- XAMPP
- Composer

## Automation

The included Selenium script:

- Opens the login page
- Generates random credentials
- Fills the login form automatically

## Note

This repository contains my automation script and documentation only. It does **not** include the original Laravel project because it was provided as part of an assignment.
